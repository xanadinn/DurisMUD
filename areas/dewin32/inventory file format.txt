In case you're wondering, here's the inventory file format:

And no, comments are not actually supported by DE



  *** DE inventory file v2    ; magic header, specifies version (v1 doesn't support descs)

  M
  356 75          ; put mob #356 in inventory, rand % is 75, implied ID is 0

  O 
  392 100         ; put obj #392 in inv, 100% rand, ID is 1

  P
  454 100 1       ; put obj #454 in ID 1, which is obj #392 loaded above, ID is 2

  E
  123 100 0 5     ; equip obj #123 to pos 5 of ID 0, which is mob #356 above, positions 
                  ; are same as .zon, ID is 3

  P
  397 100 0       ; put obj #397 in ID 0, which is mob #356 above's inventory, ID is 4

  P
  987 100 3       ; put obj #987 in ID 3, which is obj #123 equipped to mob #356 above, ID is 5

  D               ; extra desc with keywords and description - no ID
  keywords~
  description
  ~

  S               ; end of file
